# ðŸ“Š **2. What Is a Table?**

* **Orders**

# ðŸ§© **3. What Are Columns?**

A **column** represents a single attribute or category of data within a table.

Example:

* In a **Students** table, columns might be:

  * `StudentID`
  * `FirstName`
  * `LastName`
  * `Age`

* A **data type** (e.g., INT, VARCHAR, DATE)

Think of columns as **headings**.
