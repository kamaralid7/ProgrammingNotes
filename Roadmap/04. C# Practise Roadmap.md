#### Detailed Week 1: C# Language Fundamentals
This week covers basics like data types (value vs. reference) and IEnumerable, building to simple collections and memory concepts. Use console apps for all exercises.

| Day | Focus | Topics | Activities / Resources |
|-----|-------|--------|----------------------|
| **1** | C# Basics | Data types (value: int, bool, struct; reference: class, string, array), variables, var keyword, type casting, nullables | Write console app demonstrating value vs. reference assignment (e.g., copying ints vs. modifying class instances); Microsoft Learn – C# fundamentals (https://learn.microsoft.com/dotnet/csharp/fundamentals/types). |
| **2** | Control Flow & Basic Collections | If/else, switch, loops (for, while, foreach), arrays, List<T>, Dictionary<TKey, TValue> | Build a menu-driven console app with user input and array/list operations; Practice foreach on arrays. |
| **3** | Methods & Delegates | Methods (parameters, returns), delegates (Func<T>, Action<T>), custom delegates, Predicate<T> | Implement filtering with Predicate<T> (e.g., list of numbers where even); Explore lambda expressions. |
| **4** | Advanced Types | Tuples (named/unnamed), records (intro with immutability), structs vs. classes, enums | Refactor simple data holders to use tuples and records; Compare struct copying vs. class references. |
| **5** | Memory Basics | Stack vs. heap allocation, value/reference semantics, IDisposable, using statement, boxing/unboxing | Create a disposable resource class; Analyze object lifetime with using blocks; Use GC.Collect for demos. |
| **6** | Enumerables & Iteration | IEnumerable<T> (interface, GetEnumerator), IEnumerator, yield return for lazy iterators, deferred execution | Implement custom IEnumerable for a sequence (e.g., Fibonacci); Test foreach and manual enumeration. |
| **7** | Review + Mini Project | Integrate Week 1 concepts: types, flow, methods, enumerables | Build a console app processing user data (e.g., list filtering with LINQ intro); Review pitfalls like null references. |

#### Detailed Week 2: .NET Ecosystem and Advanced C#
Shift to practical .NET, including LINQ, async, and web basics. Introduce C# 14 features like field keyword in properties and extension members for type enhancements.

| Day | Focus | Topics | Activities / Resources |
|-----|-------|--------|----------------------|
| **8** | LINQ Deep Dive | LINQ basics (Select, Where, OrderBy), Join, GroupBy, Any/All, deferred vs. immediate execution | Convert loop-based code to LINQ queries; Simulate data sets for grouping; Microsoft Learn LINQ tutorials. |
| **9** | Async & Concurrency | async/await, Task<T>, cancellation tokens, thread pool, ValueTask for perf | Build async console app simulating API calls (e.g., Task.WhenAll); Handle exceptions in tasks. |
| **10** | .NET Internals & Perf | CLR overview, GC generations, memory pressure, Span<T>/Memory<T> for allocation-free ops, .NET 10 JIT improvements | Benchmark allocations with BenchmarkDotNet; Explore Span<T> in arrays; Docs on .NET 10 perf (https://devblogs.microsoft.com/dotnet/performance-improvements-in-net-10/). |
| **11** | ASP.NET Core Basics | Middleware pipeline, routing, controllers, Minimal APIs, dependency injection | Set up a basic REST API project; Add endpoints; ASP.NET Core docs (https://learn.microsoft.com/aspnet/core/). |
| **12** | Data Access | EF Core (DbContext, migrations, tracking), Include() for eager loading, basic CRUD, JSON mapping in .NET 10 | Integrate EF Core into API; Query with LINQ; Handle relationships. |
| **13** | Testing & Security | xUnit/NUnit for units, Moq for mocking, logging (Serilog), exception handling, basic auth (JWT intro) | Write tests for API endpoints; Add logging; Explore .NET 10 security defaults. |
| **14** | Wrap-Up Project & C# 14 | End-to-end app: Combine async, LINQ, EF, API; Intro C# 14 (field keyword, extension members) | Build/refactor full app; Experiment with C# 14 features in VS 2026; Identify gaps for Phase 2. |
